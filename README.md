Tetris
======

A classic Tetris game written in Java.
<br/><br/>
Special features:
<ul>
	<li>Best scores are stored on cloud</li>
	<li>Intuitive effect sounds</li>
	<li>Support Java applet to be run on web browser</li>
</ul>
